Oasis-500M Validation
=====================

1. Run inference:
   ~/.claude/commands/tools/run-test.sh --hw oasis-ttlang/oasis_inference.py

2. Copy the output frame to local:
   ~/.claude/commands/tools/remote-run.sh cat /tmp/oasis_frame.png > /tmp/oasis_frame.png

3. Open /tmp/oasis_frame.png and verify it looks like a Minecraft frame:
   - Should show a clear Minecraft scene (grass, sky, structures)
   - Should NOT be blocky, noisy, or garbled
   - Should closely match the PyTorch reference at /tmp/ref_frame.png
     (generated by triage_full.py)

Current timing (T=2, 5 DDIM steps, traced execution):
  - Per step: ~49ms
  - FPS: ~20.3
  - VAE decode: ~0.6s
